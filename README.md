# KHANHPC iOS Simulator

**KHANHPC iOS Simulator** là một trình mô phỏng iOS 14 được phát triển bằng HTML, CSS và JavaScript (jQuery). Dựa trên mô hình thực tế của iPhone 12 Pro Max, dự án này mang đến trải nghiệm iOS chân thực với giao diện trực quan và nhiều ứng dụng hoạt động đầy đủ.
